### 1、python 解释器的安装

- 1.下载安装包

   ```
   https://www.python.org/downloads/
   ```

- 2.安装

   ```
   根据安装包提示安装即可，
  安装过程勾选 
  1. Use admin privileges when installing py.exe 选项
  2. Add python.exe to PATH 选项
    
  添加环境变量等选项，也可不勾选手动配置
   ```
