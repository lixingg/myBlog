## HTML
 
### html文档的构成
- 标签
- 属性
- 文本
- 注释

### 常见标签
- 段落标签 `<p></p>`
- 标题标签 `<h1></h1>`
- 换行标签 `<br>`
- 水平线标签 `<hr />`
- 图片标签 `<img />`
- 链接标签 `<a></a>`
- 区块标签 `<div></div> <span></span>`
- 列表标签 `<ul></ul> <ol></ol> <li></li>`
- 表格标签 `<table></table> <tr></tr> <td></td>`
- 表单标签 `<form></form> <input /> <textarea></textarea> <select></select> <option></option>`
